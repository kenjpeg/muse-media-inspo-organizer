# Muse — project context

Local macOS Electron app for collecting, organizing, and searching design/image files. React 19 renderer, TypeScript everywhere, SQLite via better-sqlite3, optional Ollama (LLaVA) + Python sidecar for CLIP-style similarity.

---

## Commands

| Command | Purpose |
|---------|---------|
| `npm install` | Install dependencies |
| `npm start` | Dev: Electron Forge + Vite (renderer hot-reload; restart for main/preload changes) |
| `npm run lint` | ESLint (`*.ts`, `*.tsx`) |
| `npm run package` / `npm run make` | Forge packaging / installers |

Python (optional embedding server): see `python/requirements.txt` and `README.md`.

## Architecture (data flow)

1. **Renderer** calls `window.electronAPI.*` (`src/renderer/lib/ipc.ts`).
2. **Preload** forwards via `ipcRenderer.invoke` / `on` (`src/preload/preload.ts`).
3. **Main** handles in `ipc-handlers.ts` and `index.ts` (dialog, theme, log, drops echo).

Imported assets live under user data / app `data/` (see README): `library.db`, `originals/`, `thumbnails/` — gitignored locally.

---

## IPC surface (channel → responsibility)

Declared in preload as methods; channels in main:

**Folders:** `folders:getAll`, `folders:create`, `folders:update`, `folders:delete`

**Images:** `images:query`, `images:getById`, `images:update`, `images:trash`, `images:restore`, `images:delete`, `images:getCounts`

**Tags:** `tags:getAll`, `tags:create`, `tags:delete`, `tags:addToImage`, `tags:removeFromImage`, `tags:getForImage`, `tags:getTotalCount`

**Colors:** `colors:getForImage`

**Import:** `import:files`

**AI:** `ai:status`, `ai:autoTag`, `ai:startSidecar`, `ai:stopSidecar`, `ai:searchByText`, `ai:findSimilar`

**Misc (main/index.ts):** `dialog:openFiles`, `app:getDataPath`, `theme:get`; events `theme:changed`, `files-imported`; `clipboard:copyImage`; `ipcMain.on('log')`.

---

## Conventions / pitfalls

- **Main vs renderer:** Changing `src/main/**` or `src/preload/**` requires full app restart; renderer hot-reloads via Vite.
- **Dropped files:** Paths must come from preload (`webUtils.getPathForFile`) — not from renderer-only APIs.
- **Local images:** `local-file://` custom protocol is registered privileged and streamed from disk (`src/main/index.ts`).
- **AI:** Graceful degradation if Ollama/sidecar missing; README documents models and Python venv.

---

## When editing

Prefer extending existing repos and IPC naming patterns. Keep thumbnail/color extraction non-fatal where already wrapped. Match Zustand + component layout in `components/` rather than introducing parallel state layers unless necessary.
